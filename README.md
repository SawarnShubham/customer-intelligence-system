# Customer Intelligence & Revenue Optimization System

## 🚀 Overview
This project analyzes customer behavior and builds a system to:
- Segment customers (RFM Analysis)
- Predict churn
- Generate business insights

## 🛠 Tech Stack
- Python (Pandas, NumPy)
- SQL
- Power BI / Streamlit
- Machine Learning (Scikit-learn)

## 📊 Features
- Data Cleaning & EDA
- Customer Segmentation
- Revenue Analysis
- Interactive Dashboard
